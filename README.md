# Proto

## Getting Started

Установить зависимости и запустить сервер
```bash
git clone https://github.com/0777144/proto && cd proto
cp example.env .env
npm install
npm run server
```

or

```
yarn && yarn server
```

Запустить watcher изменений для разработки
```bash
npm run watch
```


## Features

* [webpack 2.2.1]
* JavaScript [ES6] syntax
* [React]
* [CSS Modules] and [SASS] syntax
